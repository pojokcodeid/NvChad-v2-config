Laravel Blade Snippets, Syntax Highlighting and Formatting for VS Code. 

Converted from [language-blade](https://github.com/jawee/language-blade).

## User Setting

Enable html emmet:

>```json
>"emmet.includeLanguages": {
>   "blade": "html"
>},
>```

## Features
- Syntax highlighter
- Blade snippets
- Blade formatting
